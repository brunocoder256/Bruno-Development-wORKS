# Figma Style Guide - Bruno Development Works Portfolio

## Color Palette
- **Primary Background**: Linear gradient from #0a0a0a to #1a1a1a
- **Accent Gold**: #ffd700 to #ffed4e (gradient)
- **Accent Blue**: #00bfff
- **Text White**: #ffffff
- **Text Gray**: #b0b0b0
- **Text Dark Gray**: #888

## Typography
- **Font Family**: Inter (Google Fonts)
- **Hero Title**: 4.5rem, 800 weight, gradient text
- **Section Headings**: 3.5rem, 700 weight, gradient text
- **Body Text**: 1.2rem, 400 weight, #b0b0b0
- **Small Text**: 0.9rem, 500 weight, #888

## Components

### Navigation
- Fixed header with backdrop blur
- Logo: Gradient gold text
- Links: White with gold underline animation on hover

### Hero Section
- Split layout: Text left, laptop mockup right
- Title: Large gradient text with fade-in animation
- CTA Button: Gold gradient with hover effects
- Laptop Mockup: CSS-based with floating animation

### Service Cards
- Glass-morphism effect (backdrop-filter: blur)
- Icons: Large emojis or custom icons
- Hover: Lift and background change

### Portfolio Items
- Image overlays with slide-up animation
- Gradient buttons
- Hover scale effects

### Testimonials
- Glass cards with subtle borders
- Italic quotes
- Author info below

### Contact Form
- Glass container
- Focus states with gold borders
- Animated submit button

## Spacing
- Section padding: 6rem
- Container max-width: 1400px
- Grid gaps: 2rem to 4rem
- Element margins: 1rem to 3rem

## Animations
- Fade-in up: 1s ease-out
- Float: 6s infinite
- Hover scale: 1.05
- Scroll reveal: translateY with opacity

## Breakpoints
- Desktop: >1024px
- Tablet: 768px-1024px
- Mobile: <768px
- Small mobile: <480px

## Effects
- Backdrop blur: 10px-20px
- Box shadows: Subtle dark shadows
- Gradients: Linear 135deg for all elements
- Transitions: 0.3s ease for most interactions
